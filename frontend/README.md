# frontend

Nuxt 4 com TypeScript, Pinia e SCSS.

- `app/pages`: telas protegidas do ERP.
- `app/stores`: estado, validacao de formulario e chamadas ao BFF.
- `server/api`: BFF Nitro que conversa com a API Go.
- `server/contracts`: contratos usados por BFF, testes e mocks.

Execute localmente:

```sh
pnpm install
pnpm dev
```
